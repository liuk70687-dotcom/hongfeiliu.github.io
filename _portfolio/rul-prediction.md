## Overview

This project focuses on predicting the **Remaining Useful Life (RUL)** of turbofan engines using the NASA C-MAPSS dataset.

---

## Key Contributions

- Built an **LSTM-based time-series prediction model**  
- Implemented a **DCNN variant** for comparison  
- Applied sequence windowing (30 time steps)  
- Evaluated performance using RMSE and PHM score  

---

## Results

- LSTM achieved significantly lower RMSE than DCNN  
- Demonstrated importance of temporal dependency modeling
